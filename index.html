Tapiko

Uses an iPhone’s NFC capabilities to read a Japanese PASMO or SUICA transit card and display its balance and history.

Privacy Policy

This application only reads balance and transaction history from compatible PASMO and Suica transit cards using Apple's Core NFC framework.

Information Collection, Use, and Sharing

Tapiko App does not: collect, access or record any personal information, collect, access or record any location information, collect, access or record any other information, contain any In-App purchase, display any advertising, ask for credit card number, ask for login information, ask for any personal information, lock you into subscription, send any data anywhere.

No personal information is collected, stored, transmitted, or shared.

All card reading is performed locally on the user's device.
